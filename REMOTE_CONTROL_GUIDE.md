## â“ Troubleshooting

### "Focus not visible"
- Check if blue glow appears around items
- Try pressing arrow keys multiple times
- Restart app if focus is stuck

### "Navigation not working"
- Ensure remote is paired with TV
- Check remote batteries
- Verify app is in foreground
- Try pressing OK to refocus

### "Can't select item"
- Navigate to item with arrows
- Ensure blue glow is on correct item
- Press OK firmly (not pointer click)

### "Stuck in input field"
- Press BACK button to exit
- Or press â¬‡ï¸ to move focus away

## ğŸ“ Tips & Best Practices

### For Users
1. **Use Arrow Keys First:** Navigate before pressing OK
2. **Watch for Blue Glow:** Indicates current position
3. **Press BACK to Exit:** Works in all screens
4. **Use Categories:** Faster than scrolling all channels
5. **Try Search:** Type partial name for quick find

### For Developers
1. **Test with Real Remote:** Pointer behavior differs
2. **Check Focus Indicators:** Must be visible from 10ft
3. **Verify Smooth Scrolling:** No jank or lag
4. **Test Edge Cases:** Last item, empty list, single item
5. **Validate ARIA Labels:** For screen reader support
